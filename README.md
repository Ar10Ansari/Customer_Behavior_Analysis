# Customer Behavior Analysis

## Overview
This project analyzes customer purchasing behavior using Python and data analysis techniques to identify trends and patterns in customer transactions.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Features
- Customer ID
- Age
- Gender
- Product Category
- Purchase Amount
- Review Rating

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Data Visualization
4. Insights Generation

## Key Insights
- Age group 25-40 contributes the highest revenue
- Clothing category has highest sales
- Female customers purchase more frequently
