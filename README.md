# Customer Behavior Analysis

## Project Overview
This project analyzes customer purchasing behavior to identify patterns in spending, demographics, and product preferences. The analysis helps businesses understand their customers and make better marketing decisions.

---

## Objective
The goal of this project is to:

- Understand customer purchasing patterns
- Identify high-value customer segments
- Analyze product category performance
- Generate business insights from data

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset Features

The dataset contains the following attributes:

- Customer ID
- Age
- Gender
- Item Purchased
- Category
- Purchase Amount
- Review Rating
- Payment Method
- Location

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Business Insights

---

## Key Insights

- Customers aged **25–40** contribute the highest revenue.
- **Clothing category** has the highest purchase frequency.
- Female customers purchase slightly more frequently than male customers.
- Average product rating is around **3.7–4.0**.
